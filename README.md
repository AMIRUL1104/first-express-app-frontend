# 🚀 First Express App Frontend

A simple frontend project built to interact with a **Node.js + Express backend API**.  
This project is focused on learning how frontend communicates with backend using REST API.

---

## 📌 Project Overview

This project demonstrates how a frontend application fetches and displays data from an Express server.

It helps in understanding:
- API integration
- JSON data handling
- Frontend-backend communication
- Basic project structure in a real-world workflow

---

## ⚙️ Tech Stack

- next.js
- tailwind
- Fetch API
- Express.js (Backend dependency)
- daisyUI

---

## ⚙️ How It Works

- Frontend sends request using `fetch()`  
- Express backend returns JSON response  
- Data is displayed dynamically in UI  

---

## 🎯 Learning Purpose

This project helps to understand:

- Express server setup  
- Routing system  
- JSON response handling  
- Backend workflow basics
